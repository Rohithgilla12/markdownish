Markdownish

A personal, no-fluff markdown editor for people who edit too much markdown.

Why this exists

I work a lot with agentic coding tools — Claude Code, Cursor agents, the lot — which means I'm constantly editing markdown files. CLAUDE.md, AGENTS.md, SKILL.md, specs, plans, PRDs. Opening a full IDE just to tweak a markdown file is overkill.

Scope is deliberately small. Resist the urge to add features that already exist in Obsidian or your IDE.

The stack

Tauri 2 in the shell. React, TypeScript, Vite on the inside. Tailwind for styling, shadcn/ui for primitives, and the smallest possible set of markdown plugins.

═══ NAME ════════════════════════════════════════════════

markdownish — open a folder, edit the markdown, get out.

I work a lot with agentic coding tools, which means I am constantly editing markdown files — CLAUDE.md, AGENTS.md, SKILL.md, specs, plans, PRDs. Opening a full IDE just to tweak a markdown file is overkill.

This is a tiny purpose-built tool for that workflow. Scope is deliberately small. Resist the urge.
